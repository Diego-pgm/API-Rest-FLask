EL programa prueba.py es una simulación del proyecto sin la libreria Flask en ella se crean las entidades Tienda, Producto e Inventarios 
y se hace la funcionalidad de la API donde se pueden asociar/agregar inventarios a una tienda y se puede retirar cierta cantidad de una
tienda, si no hay suficientes productos en "stock" se lanza un mensaje de error.

La carpeta templates tiene todos los códigos de html que son el diseño básico de la API

Cuando se corre el programa app.py automaticamente se crean las tablas pero en caso de fallar este se puede correr el programa crear_tablas.py

Para mejorar el código se crearon 3 clases separadas para cada Entidad en al API-Rest de Flask
